There is no phone: README
=========================

### （最后一次更新时的游戏版本：26.07.16.3）

注意！本游戏会含有一些快速闪烁画面，光敏性癫痫患者请勿游玩！

建议：将游戏放在一个单独的新建文件夹里面，这样可以避免您的关键文件被覆盖！

启动方式：双击那个`There is no phone.bat`即可运行！

（如果想以管理员权限运行，请打开隐藏的`Files`文件夹，打开里面的There is no
phone.bat，再运行，否则会提示找不到文件）
